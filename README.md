## simple-comp component library

### 安装

```bash
npm install simple-comp --save
```

### 使用

```javascript
// 加载样式
import 'simple-comp/dist/index.css'

// 引入组件
import {Button} from 'simple-comp'

const App = () => (
	<>
		<Button btnType="primary"> button </Button>
	</>
)
```

### 特性

-   🚀 最新技术栈：使用 React Hooks/typescript 等前端前沿技术开发
-   📦 开箱即用的 React 组件
